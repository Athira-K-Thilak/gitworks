>git init [to initialize]

>git status [to list]

>git add -A [to add all untracked files to staging area]

>git commit -m "commit message" [to commit added file to local repository]

>git clone [to take exact copy from remote repository to local workspace]

>git pull [to take changes from remote repository to local workspace]

>git push [to send changes from local repository to remote repository]

>git config --list [to list configuration]

>git config --global user.name "username"  [to add username]

>git config --global user.email "useremail" [to add user email]

>git remote add remote_name remote_path [to add remote]

>git push remote_name branch
